Kids Corner Guardian

Turn on Safe Browsing

Set this up once on each computer. It keeps the browser on the learning apps a grown-up allowed.

👧 Your child doesn't install anything — you set it up once here, and they can't turn it off.

🪟 Windows Your device

Chrome or Edge. No admin needed.

⬇ Download the Windows installer
  1. Open the downloaded kids-corner-guardian-install.reg.
  2. Click Yes when Windows asks.
  3. Fully close and reopen Chrome (or Edge).

That's it — the Guardian turns itself on.

🍎 Mac Your device

Chrome or Edge.

⬇ Download the Mac installer
  1. Open the downloaded profile.
  2. System Settings → General → Device Management → select it → Install (it may say "Unverified" — that's normal).
  3. Fully close and reopen Chrome.

💻 Chromebook Your device

A personal Chromebook can only install from the Chrome Web Store — that listing is in review; check back soon. A school/managed Chromebook: add it from your Google Admin console (ID jcidbnilldagoigommaalmndbgkkkbdk + this update URL: https://kids-corner-45fc2.firebaseapp.com/guardian/updates.xml).

Advanced: install without the file (any Windows/Mac browser)
  1. Download & unzip this package.
  2. Open chrome://extensions → turn on Developer mode.
  3. Click Load unpacked → pick the kids-corner-guardian folder.

(Removable, and Chrome shows a developer-mode reminder — the installer above is sturdier.)
