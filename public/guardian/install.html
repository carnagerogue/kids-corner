Kids Corner Guardian

Add Safe Browsing to Chrome

Works on Mac and Windows — about a minute, no accounts, no admin.

🛡️ Once it's added, it keeps the browser on the learning apps a grown-up allowed.

Add it to Chrome

Chrome or Edge, on Mac or Windows.

⬇ Download the extension
  1. Unzip the download. You'll get a folder named kids-corner-guardian in your Downloads. (Mac: double-click the .zip. Windows: right-click → Extract All.)
  2. Open a new tab and go to the extensions page: — tap to copy, then paste it in the address bar and press Enter.
  3. Turn on Developer mode (the switch in the top-right corner).
  4. Click Load unpacked, then choose the kids-corner-guardian folder.
✅ Go back to Kids Corner and refresh — the shield turns on.

💻 Chromebook

A personal Chromebook can only install from the Chrome Web Store (that listing is in review — check back soon). A school/managed Chromebook installs from your Google Admin console: ID jcidbnilldagoigommaalmndbgkkkbdk, update URL https://kids-corner-45fc2.firebaseapp.com/guardian/updates.xml.

Grown-ups: make it so the child can't remove it (advanced)

Force-installs it and hides the remove button. Needs a one-time setup and admin rights.

  1. Windows: download this file, open it, click Yes, restart Chrome.
  2. Mac: only reliable on a managed (MDM) Mac — on a personal Mac, Chrome ignores the profile, so use the "Load unpacked" method above instead.
